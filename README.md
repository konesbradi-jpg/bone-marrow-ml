## Medical Decision Support Application

## Description
This project builds a machine learning model to predict the success of pediatric bone marrow transplants using clinical data.

## Dataset
The dataset contains clinical information about children who underwent bone marrow transplantation.

## Project Structure
- notebooks/
- src/
- app/
- tests/
- requirements.txt

## Installation
pip install -r requirements.txt

## Run the Project
python src/train_model.py

## Run Tests
pytest tests/
